# CV Web — Badr Eddine Nassiri

CV web responsive de **Badr Eddine Nassiri**, Data Scientist & AI Engineer.

## Contenu

Reprise fidèle du CV PDF :

- Profil
- Expériences professionnelles (Teal Technology Services / OCP Group, Mamda-Mcma)
- Compétences techniques
- Projets réalisés
- Formation, langues, certifications, qualités

## Lancer en local

Ouvrir `index.html` dans un navigateur, ou servir le dossier :

```bash
python3 -m http.server 8080
```

Puis aller sur `http://localhost:8080`.

## Imprimer / exporter en PDF

Utiliser le bouton **Imprimer / PDF** en haut à droite (ou `Ctrl/Cmd + P`).
